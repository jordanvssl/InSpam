# InSpam

SpamBot Instagram [Python] for Google Chrome

## Installation

Download [chromedriver](https://chromedriver.storage.googleapis.com/index.html?path=89.0.4389.23/) for your browser.

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install InSpam.

```bash
pip install -r requirements.txt
```

## Usage

```python
python inspam.py
```
